
### 단 하나의 추상 메서드만 선언된 인터페이스

```java
interface MyFunction {
	public abstract int max(int a, int b)
}

MyFunction f =new MyFunction()
{
	public int max(int a,int b){
	return a>b ? a:b;
	}
}
int value = f.max(3,5); //해결

MyFunction f = (a,b) -> a>b ? a : b;
int value = f.max(3,5); 
```
